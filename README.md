# نظام إدارة الطوارئ الطبية 🏥 (Emergency Medical System)

نظام إلكتروني متكامل لإدارة أقسام الطوارئ في المستشفيات والمراكز الطبية. مبني باستخدام **PHP 8 (Pure MVC)** و **MySQL**، ويقدم تجربة مستخدم عصرية ومستقرة.

## 🌟 الميزات الرئيسية

- **نظام حماية وصلاحيات (RBAC):** تقسيم الأدوار (مدير، طبيب، ممرض، استقبال) وتأمين الصفحات بناءً على دور المستخدم.
- **إدارة المرضى والزيارات:** فتح ملفات طوارئ جديدة وتسجيل بيانات المرضى.
- **الفرز الطبي (Triage):** شاشة مخصصة للتمريض لتسجيل العلامات الحيوية (الضغط، النبض، الحرارة).
- **الفحص الطبي:** شاشة مخصصة للأطباء لكتابة التشخيص، خطة العلاج، والوصفات الطبية.
- **لوحة تحكم تفاعلية (Dashboard):** عرض حي للحالات النشطة في الطوارئ والأسرّة المتاحة.
- **السجل الطبي التاريخي (Timeline):** عرض تسلسلي لجميع زيارات المريض السابقة، جاهز للطباعة بضغطة زر.
- **تجربة مستخدم حديثة:** واجهة باللغة العربية (RTL) متجاوبة بالكامل مبنية بـ Bootstrap 5 مع تنبيهات SweetAlert2.

## 🛠️ التقنيات المستخدمة
- **الواجهة الخلفية (Backend):** PHP 8.2+ (MVC Architecture)
- **قاعدة البيانات:** MySQL / MariaDB
- **الواجهة الأمامية (Frontend):** HTML5, Vanilla CSS, Bootstrap 5 (RTL), FontAwesome, SweetAlert2.

## 🚀 طريقة التشغيل (التثبيت المحلي)

1. **استنساخ المستودع (Clone):**
   ```bash
   git clone https://github.com/yourusername/emergency-medical-system.git
   cd emergency-medical-system
   ```

2. **قاعدة البيانات:**
   - قم بتشغيل الخادم المحلي (مثل XAMPP أو Laragon).
   - توجد طريقتان لإعداد قاعدة البيانات:
     1. قم باستيراد ملف `database.sql` يدوياً في phpMyAdmin.
     2. **أو** افتح المتصفح على الرابط `http://localhost:8000/setup_db.php` ليتم إنشاء الجداول تلقائياً.

3. **تشغيل الخادم:**
   - يمكنك تشغيل الخادم عن طريق النقر المزدوج على ملف `start_server.bat` (في ويندوز).
   - أو تشغيله يدوياً عبر سطر الأوامر:
     ```bash
     cd emergency-medical-system
     php -S localhost:8000 -t public
     ```

4. **بيانات الدخول الافتراضية:**
   - **اسم المستخدم:** `admin`
   - **كلمة المرور:** `password`

## 📁 هيكلية المشروع
التطبيق مبني بهيكلية MVC نظيفة بدون استخدام إطارات عمل (Framework-less):
- `/app/Controllers` - المتحكمات التي تدير المنطق (Logic).
- `/app/Models` - النماذج للتعامل مع قاعدة البيانات.
- `/app/Views` - واجهات العرض (HTML/PHP).
- `/app/Core` - قلب النظام (الموجه Router، قاعدة البيانات، الجلسات).
- `/public` - نقطة الدخول الرئيسية (`index.php`) والملفات العامة (CSS/JS).
